Sistema de envio de Reportes

Solución innovadora para la creación y envío de reportes personalizados, que digitaliza y automatiza procesos manuales, evitando el uso y pérdida de papel. El envío de reportes queda registrado en el Gmail del departamento correspondiente. El proyecto fue desarrollado con React y Node.js, organizando el código en dos carpetas principales: backend (servidor y servicio de correo) y client (componentes JSX).
